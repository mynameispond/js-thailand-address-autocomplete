Example 02: Default Value + Language Switching

เคสที่ครอบคลุมในหน้านี้

  • ตั้งค่า default value ด้วย data-jsata-value
  • สลับภาษาในตอน init ด้วย jsataInit(..., { lang: "..." })
  • บังคับ render ใหม่ด้วย reinitialize: true
  • โหลดภาษา custom จาก slug ไฟล์ เช่น lib/lang/demo.json
